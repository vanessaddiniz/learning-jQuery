Selecting Elements in jQuery

By ID

              
$('#select-by-id').click(function() {
  $('#giveMeAText').text("Hello there! Did you notice I wasn't here before?");
});

We use $('#giveMeAText') to select the element with the id "giveMeAText" and then use the text() method to set its inner text to "Hello there! Did you notice I wasn't here before?"

By Class

              
$('#select-by-class').click(function() {
  $('.text-info').text('spans with class "text-info" changed text');
//using objects to change multiple properties
  $('.text-info').css({
    textDecoration: 'none',
    fontSize: '2rem'
  });
});

Hi there! So, this time i have a span with a class called blue here and here, but not here. I think you already know what to do!

We use $('#.text-info') to select the elements with the class "text-info" and then use the text() method to set its inner text to "spans with class "text-info" changed text" and the css() method to remove the underlines by changing its text-decoration to 'none'
