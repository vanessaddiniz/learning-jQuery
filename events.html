Events

multiple events

It is possible to set the same event to multiple elements at once

              
$('.sameEvent').click(function() {
$(this).removeClass('btn-lilac');
$(this).addClass('btn-danger');
});

This code is used to add an event (the function described) that is going to trigger when we press the button above. The function tells the browser to display the paragraph that is now hidden and its content

on

The on() method is just like addEventListener in vanilla JS.It lets you specify what type of event you're adding

On('click') vs click()

click() only adds listeners for existing elements

on() will add listeners for all future potential elements

On double click

          
$('#doubleClickMe').on("dblclick", function() {
  alert("You've clicked me, I've got it already!")
});
